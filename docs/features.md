# List of Features

### Platforms

- Mac
- Linux
- Server

### Interface

- Tabs
    - Panels
        - Split
        - Different layouts
        - Drag and drop tabs to rearrange
    - Navigation:
        - Use Alt-Tab and Alt-Shift-Tab to go next and previous


### Code Editor

- Syntax highlighting
- ACE editor
- Autocompletion
    - On keypress
    - Specifics backends: Ctags, Doks, ...

### Files

- Lateral files explorer
    - Files tree
    - Automatic updates

### Terminal

- Complete terminals
- Collaborative terminals
